# Część 1 — Wygląd (Frontend)

## Technologia

- **Framework:** React.js
- **Styling:** Tailwind CSS
- **Routing:** React Router

## Struktura stron

### 1. Strona główna (Landing Page)
Cel: przekonanie nowego użytkownika do rejestracji. Zawiera:
- Nagłówek z nazwą platformy i przyciskiem CTA (Generuj dokumenty)
- Trzy karty z zaletami: 9 typów dokumentów / Gotowe w 30 sekund / UE i poza UE
- Sekcja jak to działa — 4 kroki z ikonami
- Cennik — trzy plany (abonament / per dokument / prowizja)
- Stopka z linkami

### 2. Rejestracja i logowanie
Dwie osobne strony. Rejestracja zbiera: nazwa firmy, email, hasło, NIP (opcjonalnie).
Po rejestracji użytkownik trafia do dashboardu. Logowanie standardowe email + hasło z opcją Reset hasła.

### 3. Dashboard (strona główna po zalogowaniu)
- 3 metryki na górze: dokumenty w tym miesiącu / aktywny plan / zaoszczędzony czas
- Przyciski szybkich akcji: Nowy dokument / Historia / Moje firmy
- Tabela ostatnich dokumentów: typ dokumentów / trasa / data / przycisk PDF

### 4. Formularz generowania dokumentów (4 kroki)

| Krok | Co robi użytkownik |
|------|--------------------|
| Krok 1 — Typ i trasa | Wybiera: Drogowy (TIR) lub Morski. Wpisuje kraj nadania i kraj docelowy. System automatycznie wykrywa czy trasa jest poza UE i wyświetla komunikat o dokumentach celnych. |
| Krok 2 — Towar | Wpisuje: opis towaru, wagę w kg, objętość w m3, wartość w EUR/CHF/USD, ilość paczek. Opcjonalnie: kod celny CN/HS (dla tras poza UE). |
| Krok 3 — Dane firm | Wpisuje dane nadawcy i odbiorcy: nazwa firmy, adres, kraj, NIP/VAT. Możliwość zapisania firmy do bazy. |
| Krok 4 — Generuj | Przegląd wszystkich danych. Lista dokumentów które zostaną wygenerowane. Przycisk Generuj. Po kliknięciu: spinner ładowania, potem lista PDF-ów do pobrania. |

### 5. Ekran wynikowy po generowaniu
- Lista wygenerowanych dokumentów z ikonami
- Przycisk Pobierz wszystko (ZIP) i osobne przyciski dla każdego pliku
- Przycisk Wygeneruj ponownie z innymi danymi
- Przycisk Zapisz zestaw dokumentów do historii

### 6. Historia dokumentów
- Tabela z wszystkimi wygenerowanymi zestawami
- Filtrowanie po dacie, typie transportu, trasie
- Możliwość ponownego pobrania dowolnego dokumentu
- Eksport historii do CSV / Excel

### 7. Moje firmy
- Lista zapisanych firm (nadawców i odbiorców)
- Formularz dodawania nowej firmy
- Edycja i usuwanie istniejących
- Przy generowaniu dokumentów — wybór z listy zamiast ręcznego wpisywania

### 8. Panel abonamentu
- Aktualny plan i data odnowienia
- Historia płatności
- Zmiana planu (upgrade / downgrade)
- Integracja ze Stripe

## Nawigacja

| Element | Zawartość |
|---------|-----------|
| Pasek górny (nav) | Logo platformy, linki do sekcji, przycisk Zaloguj / konto użytkownika |
| Menu boczne (po zalogowaniu) | Dashboard, Nowy dokument, Historia, Moje firmy, Abonament, Ustawienia |
| Breadcrumbs | Na formularzu 4-krokowym — widoczny pasek postępu |
| Responsywność | Pełna obsługa mobile — menu hamburger na małych ekranach |

## Komponenty UI wielokrotnego użytku

- `DocumentCard` — karta dokumentu z ikoną, nazwą i przyciskiem pobierz
- `StepProgress` — pasek kroków 1-2-3-4 z aktywnym stanem
- `TransportChip` — przycisk wyboru TIR / Morski
- `CountrySelect` — wyszukiwarka krajów z flagami
- `AlertBox` — komunikat o dokumentach celnych
- `MetricCard` — karta z liczbą i etykietą (do dashboardu)
